# ObsidianVault

Personal knowledge base built in Obsidian, versioned with Git.

## Structure

| Folder | Contents |
|---|---|
| `Java/` | Intro and Intermediate Java course notes |
| `SQL/` | PostgreSQL course notes |
| `Web/` | HTML, CSS, JavaScript, Bootstrap, TypeScript, React |
| `Spring/` | Spring Boot, JPA, Flyway |
| `Git/` | Version control notes |
| `TDD/` | Test-driven development |
| `General Skills/` | Shell, shortcuts, cross-cutting skills |
| `Self Study/` | Udemy Java, study priorities |
| `decisions/` | Architecture Decision Records (ADRs) |
| `projects/` | Active project notes |
| `daily/` | Session logs |
| `templates/` | Note templates |

## Conventions

- Notes use `[[Wikilinks]]` for cross-referencing
- Each note ends with a `## Related` section
- ADRs follow Context / Decision / Consequences format
- Obsidian Git auto-commits every 15 minutes

## Stack

Java · Spring Boot · PostgreSQL · JavaScript · HTML/CSS · Git
