## Key Concepts to Master

- **OOP pillars:** Encapsulation, Inheritance, Polymorphism
- **Method resolution:** overloading vs overriding, dynamic dispatch
- **Memory model:** pass by value (primitives) vs pass by reference (objects/arrays)
- **Complexity:** O(n), O(n²), O(n log n) — know the difference
- **Access modifiers:** public, protected, package-private, private
