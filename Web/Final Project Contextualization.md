- Build a full-stack CRUD Application using React, MUI, Spring Boot, and PostgreSQL
- No Authentication Required
    - Focus on functionality, UI, and clean code
- Graded on creativity and technical implementation

PostgreSQL:

- You must have at least two joined tables
- Server should query joined data and return to the frontend

Spring Boot:

- Should be clean, modular, and serve your CRUD endpoints

CRUD - Show It All

- Create, Read, and Update as expected, but…
- Delete: Remove data, soft delete (optionally: log it)

Forms - Be Intentional

- Should include:
    - Text or Password Input
    - Radio/Checkbox Input (2+ options)
    - File Upload or Select Dropdown with 3+ items

Frontend - React and MUI Power

- Use meaningful, styled MUI components - not just plain HTML
- Structure your components in a modular, reusable way
- Handle data gracefully from the backend

UI & UX - Polish It

- Responsiveness Matters: it should look great on mobile and desktop
    - Doesn’t need to be perfect
- All buttons, links, and inputs should work - no dead ends
- Design should be clean and intuitive
- All images should have alt tags

Stretch Goals and Extra Points

- Polish goes a long way: animations, tooltips, etc.
- Beyond MVP? Add creative features or integrate an external API
- Good commit history + clear messages on GitHub

Code & Presentation

- Code should be organized and readable: comment important parts
- ==<u>***You’ll present your app: be ready to explain structure, design choices, and learnings***</u>==
- Total points: 200 (check rubric for breakdown)
