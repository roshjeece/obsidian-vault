Using logical AND operator: `&&`

This is and operator

`||`

This is OR operator

Example

```java
public class Main {
    public static void main(String[] args) {

        int x = 3 ;
        int y = 4 ;

        if ((x > 2) || (y < 3))
            System.out.println("hello");
    }
}
```

Output: hello
