## `git branch` (tells you what branches you have access to)

- `git branch CSS` (creation of new branch named CSS)
- `git branch checkout CSS` (switching to that new CSS branch)
- `git checkout main` (switching back to main)
- `git merge CSS` (merging CSS to main [which you’re currently on])

Main Branch:

- Working code, milestone

Other Branch

- Working on new component
- Experimentation

See remote branches: `git branch -r`

Delete local branch: `git branch -d test`

Delete remote branch: `git push <remote_name> --delete [branch name]`

- `<remote_name>` is typically origin

See remote repositories: `git remote -v`
